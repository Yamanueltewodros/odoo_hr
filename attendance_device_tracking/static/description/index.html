Attendance Device Tracking

Track Employee Location & Device Information

Enhanced attendance tracking with GPS location and device information capture

Key Features

GPS Location Tracking

Automatically capture GPS coordinates when employees check in or out. View exact locations on Google Maps with a single click.

Device Information

Record IP address, browser, and device details for each attendance entry. Perfect for security and compliance requirements.

Multi-Mode Support

Kiosk Mode

  • Public kiosk interface
  • Barcode scanning support
  • PIN protection available
  • Automatic location capture

Systray Mode

  • Quick check-in from menu
  • GPS permission request
  • Real-time tracking
  • User-friendly interface

Location Intelligence

Automatic Geocoding

GPS coordinates are automatically converted to human-readable location names. IP-based location as fallback when GPS is unavailable.

Google Maps Integration

One-click access to view check-in and check-out locations on Google Maps. Perfect for verifying employee attendance locations.

Privacy & Security

  • Company-wide enable/disable control
  • No tracking for manual entries
  • Browser permission required for GPS
  • Secure data storage
  • Standard Odoo access controls

Easy Configuration

Enable device tracking with a single checkbox in HR Settings. No complex configuration required.

Settings → HR Settings → Attendances → Device & Location Tracking

Perfect For

  • Remote Teams: Verify work-from-home attendance
  • Field Workers: Track on-site presence
  • Multiple Locations: Monitor attendance across offices
  • Compliance: Meet regulatory requirements
  • Security: Enhanced audit trails

Technical Specifications

Requirements

  • Odoo 18.0

  • base_geolocalize module

Browser Support

  • Modern browsers with Geolocation API
